# Nexero Capital

This repository contains a Vite + React + TypeScript SPA for Nexero Capital.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

Deployment

This project is configured for static hosting (Cloudflare Pages, Vercel, Netlify).
See `CLOUDFLARE_DEPLOY.md` for Cloudflare Pages-specific instructions.

Environment

- Copy `.env.example` to `.env.local` when developing locally and set `VITE_ADMIN_EMAIL` and `VITE_ADMIN_PASSWORD` if desired.
